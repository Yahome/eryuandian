# T-005 Project Wiki Viewer

## 目标
在 Dev OS Dashboard 中提供项目 Wiki 快速浏览入口，让用户可以从界面查看核心 Markdown 事实源。

## 页面模块
- Wiki 文件列表
- 事实源入口
- 最近更新
- Markdown 预览或外链打开

## 验收标准
- [ ] 能看到 README、Roadmap、Task Board、Decisions、Risks
- [ ] 能看到 Agent 进度文档入口
- [ ] 能看到任务文档入口
- [ ] 不修改 Markdown 事实源内容
- [ ] 与 Dashboard UI 风格一致
