# PROJECT_BRIEF：两元店

状态：草稿

## 项目名称

中文名：两元店
英文目录名：eryuandian

## 产品定位

两元店是一个 Web App，核心是 AI 内容生成，首期包含两个业务入口：

1. 生图：面向本地商家，生成营销、活动、宣传类图片。
2. 生成试卷：面向老师、家长、学生，根据年级、科目、教材版本、单元/考试类型生成试卷。

## 首期必须覆盖

- PC 端
- 移动端
- 手机号注册/登录
- 生图工作台
- 试卷生成工作台
- 生成记录
- 会员/额度能力预留

## 开发原则

- 先建立文档事实源，再做 Dashboard。
- Dashboard 数据不得硬编码示例数字，必须来自 `docs/project-os` 的 Markdown / JSON。
- 多 Agent 分工开发，所有进度必须写回各自的 Markdown。
